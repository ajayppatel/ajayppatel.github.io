# Ajay Patel — Personal Website

My personal website at [ajayppatel.com](https://ajayppatel.com).

Built with vanilla HTML, CSS, and JavaScript. No build step required.

## Local Development

Just open `index.html` in your browser, or use any static file server:

```bash
# Python
python3 -m http.server 8000

# Node
npx serve .
```

## Deployment

Deployed via GitHub Pages. Push to `main` and it auto-deploys.
